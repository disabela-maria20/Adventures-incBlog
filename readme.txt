Olá, seja bem-vindo ao Teste de Front-End Adventures Inc

Esta pasta deverá conter, além desse arquivo readme.txt:

- logo.png - Placeholder da logo de um blog fictício.
- main.css - Estilos pré-definidos.

- index.html - A estrutura da página.
- style.css - Use este arquivo para adicionar os estilos que desejar.
- script.js - Use este arquivo para adicionar os scripts que desejar.

Fica proibido para esse teste o uso de qualquer framework e/ou biblioteca de Javascript ou CSS (ex.: JQuery, Bootstrap, React).
Fique à vontade para consultar qualquer site ou fórum na internet, caso sinta a necessidade.

#####################################
############# A TAREFA ##############
#####################################

O script em "script.js" está recebendo de forma assíncrona os posts de um blog de Radioproteção via uma API Rest.
Você deverá criar um conjunto de "post-cards" na página para exibir as seguintes informações de cada um dos posts recebidos pela API.

- Título (title)
- Resumo (excerpt)
- Data de Publicação (date)
- Autor (_embedded.author)
- Imagem de Capa (_embedded.wp:featuredmedia)
- Botão ou link para a leitura do post completo (link)

Estilize os post-cards da forma que desejar, levando em consideração a usabilidade. A página deverá ser responsiva.
